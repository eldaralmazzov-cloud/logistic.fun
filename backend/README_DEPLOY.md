# Инструкция по деплою Бэкенда (Render.com)

**Render** — это популярный сервис для бесплатного деплоя. Он не требует привязки карты для бесплатного тарифа.

### Шаг 1: Подготовка
1. Убедитесь, что ваш код загружен на GitHub.
2. Создайте аккаунт на [Render.com](https://render.com/).

### Шаг 2: Создание Web Service
1. В панели Render нажмите **New** -> **Web Service**.
2. Подключите свой GitHub-репозиторий.
3. Дайте сервису имя.
4. В разделе **Runtime** выберите **Docker**.
5. Убедитесь, что выбран тариф **Free**.

### Шаг 3: Настройка переменных (Environment Variables)
1. В настройках сервиса перейдите в раздел **Environment**.
2. Нажмите **Add Environment Variable** и добавьте:
   - `SECRET_KEY`: любая случайная строка.

### Шаг 4: Ссылка на API
Render выдаст вам адрес вида `https://your-app.onrender.com`. 
Этот адрес нужно вставить в настройки фронтенда (`frontend/.env`).

---

## Что нужно знать о бесплатном тарифе Render:
- **Спящий режим**: Если к серверу никто не обращается 15 минут, он "засыпает".
- **Пробуждение**: Первый запрос после сна может занять 30-50 секунд. Это нормально для бесплатного тарифа.
- **Порт**: Docker-контейнер настроен на порт **8000**. Render обычно сам находит его, но если возникнут проблемы, можно добавить переменную окружения `PORT=8000`.

## Решение проблемы с пропаданием данных (Supabase)

По умолчанию приложение использует SQLite, который на Render стирается каждый день. Чтобы данные сохранялись навсегда, используйте бесплатную базу данных **Supabase**.

### Шаг 1: Создание базы данных в Supabase
1.  Зарегистрируйтесь на [Supabase.com](https://supabase.com/).
2.  Создайте новый проект (Project).
3.  В боковом меню выберите **Project Settings** (шестеренка) -> **Database**.
4.  Найдите раздел **Connection string**, выберите вкладку **URI**.
5.  Скопируйте строку вида: `postgresql://postgres.[ID]:[PASSWORD]@aws-0-eu-central-1.pooler.supabase.com:5432/postgres`
    *   *Замените `[PASSWORD]` на пароль, который вы указали при создании проекта.*

### Шаг 2: Подключение к Render
1.  Зайдите в панель управления **Render**.
2.  Выберите ваш **Web Service** (бэкенд).
3.  Перейдите в раздел **Environment**.
4.  Добавьте новую переменную окружения:
    *   Key: `DATABASE_URL`
    *   Value: `строка_из_supabase_которую_вы_скопировали` (та самая, которую вы скопировали из Supabase URI).
5.  Нажмите **Save Changes**.

После этого Render перезапустит сервер, и данные будут сохраняться в Supabase навсегда.
